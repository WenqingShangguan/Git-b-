# Git-B站 wupeiqi

- 新建一个文件
  
  `touch test.py`
  
- 初始化

  `git init`

- 查看状态

  `git status`

- 保存到暂存区

  `git add filename/.`

- 提交版本

  `git commit -m 'description of this version'`

- 提交成功的前提是设置人名，邮箱
  
````
  git config --global user.name 'wenqing'
  git config --global user.email 'wenqing.shangguan@example.com'
````
- 查看提交版本记录
  `git log`
